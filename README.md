# 📚 Study Schedule Generator

A smart and interactive web application that automatically creates a personalized study timetable based on subjects, study hours, exam timeline, and priority topics.

---

## 🚀 Features

* Generate day-wise study schedules instantly
* Supports multiple subjects
* Priority-based subject allocation
* Detailed study topics and practice questions
* Expandable/collapsible study sessions
* Smart exam preparation tips
* Responsive and clean UI design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 Project Structure

```bash
├── index.html
├── scheduler.js
└── README.md
```

---

## 📌 How It Works

1. Enter subjects separated by commas
2. Select:

   * Number of study days
   * Study hours per day
   * Days remaining for the exam
3. Add priority subjects (optional)
4. Click **Generate my schedule**
5. The app creates a detailed study plan automatically

---

## 🧠 Smart Scheduling Logic

The scheduler:

* Distributes subjects evenly across available days
* Prioritizes difficult subjects if specified
* Assigns predefined topics, study tasks, and practice questions
* Gives revision tips based on exam proximity
